# Gimmick Helper

![alt text](images/image.png)

Web-based gimmick transform helper for osu!.

## 🌐 Web App

👉 https://calmeel.github.io/gimmick-helper/

---

## 🛠 Features

- `.osu` file transform tool
- Preview before download
- Difficulty suffix support
- Detach Barline transform
- SV / volume / kiai inheritance
- Supports circle / slider / spinner
- Browser-only processing
